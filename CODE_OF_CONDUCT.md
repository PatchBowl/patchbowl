# Code of Conduct

Patchbowl adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be kind. Assume good intent.
- Disagree on ideas, not people.
- No harassment, no slurs, no personal attacks. We will enforce this.
- If you see something off, email **conduct@patchbowl.dev** (replace with the real address once domain DNS is live).

## Enforcement

Maintainers may warn, temporarily mute, or permanently ban contributors who break this code in any Patchbowl space (GitHub, Discord, anywhere with the Patchbowl name on it). We aim to be fair, but our priority is keeping the community healthy for everyone else.
